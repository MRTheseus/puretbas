Privacy User Guide

pureTabs

A minimalist new tab extension that prioritizes your privacy and data sovereignty.

Privacy Policy

"Privacy by design. Data sovereignty by choice."

Zero Data Collection

This extension contains no tracking scripts, no analytics, and no advertising code.

Transparent Permissions

We only request the minimum permissions required for basic functionality (e.g., storage to save your icon layouts). We promise: past, present, and future, we will never integrate any third-party tracking scripts.

Data Sovereignty

All your data (icons, layouts, settings) exists only in two places: your local browser and your own Cloudflare D1 database. The developer has no access to your data; we don’t even know who is using the extension.

User Guide

1 Icons & Layout

  • Add Icons Click the Gear icon in the top-left (or top-right) corner to enter the Control Center. In the "Add" tab, you can create new icons with Auto-fetch or Custom design.
  • Edit Icons Right-click any existing icon on the home screen and select "Edit" to modify its name, link, or logo.
  • Layout & Background In the Control Center, you can precisely customize icon size, spacing, and grid layout. Supports uploading local wallpapers as your background.

2 Advanced Folder Operations

Add to Folder

Method A Simply drag and drop an icon onto a folder icon.

Method B Open a folder and click the "+" button inside to add icons.

Remove from Folder

Method A Open the folder and drag an icon out of the folder popup area.

Method B Right-click an icon inside the folder and select "Move out".

Dismiss Folder: Right-click a folder icon and select "Dismiss," or it will automatically dissolve when the last icon is moved out.

Note: When a folder is dismissed, the icons inside will return to the home screen and will not be deleted.

3 Sync & Backup

Private Cloud Sync

Supports cross-device synchronization via your own Cloudflare Worker + D1 database. For detailed instructions, please refer to the "Cloudflare D1 Setup Guide" inside the settings page.
